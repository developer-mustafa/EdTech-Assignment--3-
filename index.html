ফ্লেক্স বক্সঃ- ফ্লেক্সবক্স হচ্ছে সিএসএস(৩) এর নতুন লেআউট। পেজ লেআউট যখন বিভিন্ন সাইজের স্ক্রিন এবং বিভিন্ন ডিভাইসে প্রদর্শিত হবে তখন এলিমেন্ট গুলোকে পূর্ব নির্ধারিত নকশা অনুযায়ী প্রদর্শন করানোর জন্য সিএসএস(৩) ফ্লেক্সবক্স ব্যবহার করা হয়। ফ্লেক্স কন্টেইনার এবং ফ্লেক্স কন্টেন্ট নিয়ে গঠিত হয় ফ্লেক্সবক্স। একটি কন্টেইনারকে ফ্লেক্সে রূপান্তর করতে উক্ত এলিমেন্টের display প্রোপার্টিতে ভ্যালু হিসেবে flex অথবা inline-flex সেট করতে হয়। একটি ফ্লেক্স কন্টেইনারের মধ্যে এক বা একাধিক ফ্লেক্স কন্টেন্ট থাকতে পারে। ফ্লেক্স কন্টেন্ট ফ্লেক্স কন্টেইনারের ভিতরে লাইনে অবস্থান করে।

গ্রিড-ভিউঃ- একটি ওয়েব পেজকে কলামে বিভক্ত করে লে-আউট তৈরি করা যায়, একেই গ্রিড-ভিউ বলা হয়।ওয়েব পেজ ডিজাইনের সময় গ্রিড ভিউ ব্যবহার করা খুবই গুরুত্বপূর্ণ। এটি ওয়েব পেজের মধ্যে এলিমেন্টকে সঠিক জায়গায় স্থাপন করা সহজ করে তোলে।রেস্পন্সিভ গ্রিড-ভিউতে অধিকাংশ সময় ১২ টি কলাম থাকে, ১২ টি কলাম মিলে এর মোট প্রস্থ হয় ১০০% এবং আপনি ব্রাউজারের আকার পরিবর্তন করার সাথে সাথে ইহা সংকুচিত ও প্রসারিত হবে।প্রতিটি সারির জন্য একটি এলিমেন্ট ব্যবহার করতে হবে। প্রতিটি সারির ভিতরে কলাম সর্বোচ্চ ১২ পর্যন্ত যোগ করা যায়

Tailwind CSS Framework :- Tailwind CSS হল দ্রুত এবং দক্ষতার সাথে কাস্টম ইউজার ইন্টারফেস তৈরির জন্য একটি ইউটিলিটি Fast CSS ফ্রেমওয়ার্ক। এটি একটি ইনলাইন স্টাইলিং যা CSS-এর জন্য কোড না লিখে একটি Clean ইন্টারফেস তৈরি করতে ব্যবহৃত হয়। টেলউইন্ড সিএসএস প্রথম ইউটিলিটি-প্রথম সিএসএস লাইব্রেরি নয়, তবে এটি সবচেয়ে জনপ্রিয় এবং হালকা একটি। এটি একটি অত্যন্ত কাস্টমাইজযোগ্য, নিম্ন-স্তরের CSS ফ্রেমওয়ার্ক, এবং এটি যেকোন ওয়েবসাইটের জন্য একটি দুর্দান্ত ইন্টারফেস তৈরি করতে প্রয়োজনীয় সমস্ত বিল্ডিং ব্লক সরবরাহ করে।

Bootstrap CSS Framework :- এটি একটি ওপেন-সোর্স ফ্রেমওয়ার্ক যা প্রতিক্রিয়াশীল ওয়েবসাইটগুলিকে আরও ভাল, দ্রুত এবং সহজে ডিজাইন করতে ব্যবহৃত হয়৷ এটি একটি শিক্ষানবিস-বান্ধব এবং সবচেয়ে জনপ্রিয় ওপেন সোর্স ফ্রেমওয়ার্ক যাতে HTML, CSS, এবং JavaScript অন্তর্ভুক্ত থাকে। বুটস্ট্র্যাপ জাভা, পিএইচপি ইত্যাদির মতো সার্ভার-সাইড প্রযুক্তির সাথে ওয়েব অ্যাপ্লিকেশন তৈরি করতে ব্যবহার করা যেতে পারে এবং এর প্রতিক্রিয়াশীল ডিজাইন মোবাইল ফোন, ট্যাবলেট এবং কম্পিউটারের মতো প্ল্যাটফর্মকে অনুমতি দেয়। বুটস্ট্র্যাপে টাইপোগ্রাফি, ফর্ম, বোতাম, নেভিগেশন, আইকন এবং অন্যান্য ইন্টারফেস উপাদানগুলির জন্য CSS এবং জাভাস্ক্রিপ্ট-ভিত্তিক ডিজাইন টেমপ্লেট রয়েছে। এটি অবজেক্ট-ওরিয়েন্টেড CSS এর উপর ভিত্তি করে। বুটস্ট্র্যাপ দ্রুত ওয়েবসাইট টেমপ্লেট ডিজাইন ও বিকাশ করতে সাহায্য করে।

CSS Box Model:

সকল এইচটিএমএল এলিমেন্টকে বক্স হিসেবে বিবেচনা করা যেতে পারে। আমরা যখন সিএসএসের মাধ্যমে ওয়েবপেজের ডিজাইন এবং লেআউট নিয়ে কথা বলি, তখন একটি টার্ম আমাদের কাছে চলে আসে তা হলো "box model"। বক্স মডেল স্বাভাবিকভাবেই একটি বক্স যা প্রত্যেকটি এইচটিএমএল এলিমেন্টকে একটি বক্সের মধ্যে মুড়িয়ে দেয়। ইহা মার্জিন, বর্ডার, পেডিং এবং প্রকৃত কন্টেন্ট নিয়ে গঠিত হয়।

এলিমেন্টের সম্পূর্ণ প্রস্থ = কন্টেন্টের প্রস্থ + বাম পাশের প্যাডিং + ডান পাশের প্যাডিং + বাম পাশের বর্ডার + ডান পাশের বর্ডার + বাম পাশের মার্জিন + ডান পাশের মার্জিন কন্টেন্ট - যা বক্সের কন্টেন্ট, যেখানে টেক্সট এবং ছবি প্রদর্শিত হয়। প্যাডিং - যা কন্টেন্টের চারপাশে ফাঁকা অংশের সৃষ্টি করে এবং ইহা স্বচ্ছ। বর্ডার - যা প্যাডিং এবং কন্টেন্টের চতুর্দিকে থাকে। মার্জিন - যা বর্ডারের বাহিরে ফাঁকা অংশের সৃষ্টি করে। প্যাডিং এর মত ইহাও স্বচ্ছ। আমরা বক্স মডেল ব্যবহার করেও একটি এলিমেন্টের চারপাশে বর্ডার সেট করতে পারি এবং এলিমেন্টের মধ্যে স্পেস(space) রাখতে পারি।

Tailwind CSS Framework :- Tailwind CSS হল দ্রুত এবং দক্ষতার সাথে কাস্টম ইউজার ইন্টারফেস তৈরির জন্য একটি ইউটিলিটি Fast CSS ফ্রেমওয়ার্ক। এটি একটি ইনলাইন স্টাইলিং যা CSS-এর জন্য কোড না লিখে একটি Clean ইন্টারফেস তৈরি করতে ব্যবহৃত হয়। টেলউইন্ড সিএসএস প্রথম ইউটিলিটি-প্রথম সিএসএস লাইব্রেরি নয়, তবে এটি সবচেয়ে জনপ্রিয় এবং হালকা একটি। এটি একটি অত্যন্ত কাস্টমাইজযোগ্য, নিম্ন-স্তরের CSS ফ্রেমওয়ার্ক, এবং এটি যেকোন ওয়েবসাইটের জন্য একটি দুর্দান্ত ইন্টারফেস তৈরি করতে প্রয়োজনীয় সমস্ত বিল্ডিং ব্লক সরবরাহ করে।

Bootstrap CSS Framework :- এটি একটি ওপেন-সোর্স ফ্রেমওয়ার্ক যা প্রতিক্রিয়াশীল ওয়েবসাইটগুলিকে আরও ভাল, দ্রুত এবং সহজে ডিজাইন করতে ব্যবহৃত হয়৷ এটি একটি শিক্ষানবিস-বান্ধব এবং সবচেয়ে জনপ্রিয় ওপেন সোর্স ফ্রেমওয়ার্ক যাতে HTML, CSS, এবং JavaScript অন্তর্ভুক্ত থাকে। বুটস্ট্র্যাপ জাভা, পিএইচপি ইত্যাদির মতো সার্ভার-সাইড প্রযুক্তির সাথে ওয়েব অ্যাপ্লিকেশন তৈরি করতে ব্যবহার করা যেতে পারে এবং এর প্রতিক্রিয়াশীল ডিজাইন মোবাইল ফোন, ট্যাবলেট এবং কম্পিউটারের মতো প্ল্যাটফর্মকে অনুমতি দেয়। বুটস্ট্র্যাপে টাইপোগ্রাফি, ফর্ম, বোতাম, নেভিগেশন, আইকন এবং অন্যান্য ইন্টারফেস উপাদানগুলির জন্য CSS এবং জাভাস্ক্রিপ্ট-ভিত্তিক ডিজাইন টেমপ্লেট রয়েছে। এটি অবজেক্ট-ওরিয়েন্টেড CSS এর উপর ভিত্তি করে। বুটস্ট্র্যাপ দ্রুত ওয়েবসাইট টেমপ্লেট ডিজাইন ও বিকাশ করতে সাহায্য করে।
